Para los sujetos que se acogen a una salida alterna, se le creará un cuaderno de extremos. Por ejemplo, si “Juan” y “José” se acogen al Principio de oportunidad, y “Luis” se acoge al Acuerdo reparatorio, entonces se crearán dos cuadernos de extremos. Si todos se acogen a una salida alterna, solo se creará un cuaderno de extremos si existen dos salidas externas diferentes, si no todo se manejará en la carpeta principal.
